# `14` equalsTen

## 📝 Instrucciones

Escribe una función llamada "equalsTen".

Dado un número, "equalsTen" retorna si el ese número es o no igual a 10.

```Javascript
var output = equalsTen(9);
console.log(output); // --> falso
```
