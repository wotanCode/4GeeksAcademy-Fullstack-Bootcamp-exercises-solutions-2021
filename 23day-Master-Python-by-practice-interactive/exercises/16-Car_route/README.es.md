# `16` Ruta de un vehículo

## 📝 Instrucciones:

Un vehículo puede cubrir una distancia de N kilómetros por día. ¿Cuántos días le tomará cubrir una ruta de M kilómetros de longitud? 
El programa recibe dos números: N y M.

**Ejemplo de entrada**
* 700
* 750

**Ejemplo de salida**
* 2
